# 人工智能通识：理解、使用与共创

面向大一学生的单页互动课程网站。课程包含 16 周、32 课时、8 个模块、4 个本地互动实验、考核与综合项目说明。核心课程内容只从 `src/data/course-data.json` 读取。

## 技术栈

- Vite + 原生 TypeScript + 语义化 HTML + 现代 CSS
- Three.js 0.185.0（桌面端课程星图，独立延迟加载）
- DOM 2D 星图（移动端、减少动态偏好或 WebGL 不可用时自动启用）
- Vitest（数据、筛选、进度与实验计算测试）
- localStorage（仅本设备学习进度，无账号、后台、数据库或外部 AI API）

## 安装与运行

```bash
npm install
npm run dev
```

浏览器打开终端提示的本地地址。

## 构建与预览

```bash
npm run build
npm run preview
```

构建结果位于 `dist/`，可以部署到任意静态托管服务。

## 测试

```bash
npm test
```

测试会验证 32 课完整性、编号与 ID、每周课时、模块引用、能力维度、递进层级、考核权重、本地进度清洗、组合筛选和互动实验核心计算。

## 课程数据维护

唯一课程数据文件是 `src/data/course-data.json`。维护课程时请直接更新该文件，不要在组件中复制课时文本。页面启动与测试都会校验：

- 恰好 32 课、编号 1–32 连续且 ID 唯一；
- 每周恰好 2 课、全部 8 个模块有效；
- 能力维度和递进层级有效；
- 总评权重合计 100%。

如果版本升级造成课时 ID 变化，本地进度读取时会自动忽略已不存在的 ID。

## 无障碍与降级

- 语义化区块、跳转主内容、清晰焦点、44px 触控目标；
- 原生 dialog、Escape 关闭、返回触发点、上一课/下一课；
- 课程星图有完整 DOM 等价入口，Canvas 不承担唯一信息；
- 拖拽分类实验同时支持“选择任务 → 选择路线”的键盘流程；
- 遵循 `prefers-reduced-motion`，小于 768px 自动使用 2D 星图；
- 375px 宽度下无页面级横向滚动，时间轴改为纵向。

## 隐私

站点不加载统计脚本，不发送课程进度，不使用外部 AI 服务。完成状态只保存在当前浏览器的 `ai-general-course-progress-v1` 键中。
